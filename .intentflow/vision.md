# EvorIDE — vision

An open, **agent-agnostic** desktop IDE for running many coding agents at once —
Claude Code, Codex, or any CLI — across all your projects, from one window, and
**always knowing which agent needs you**.

## Principles
- **Agent-agnostic.** If it runs in a shell, EvorIDE can host it. No lock-in.
- **Project-centric.** Each project collects its agents, run config, git, tasks,
  and intent in one place.
- **Intent-first.** The *why* lives in the repo (this `.intentflow/`), attributed
  to the person and the agent/model used — so it survives.
- **Local-first, fast.** Native (Tauri), real ptys, SQLite; cloud is opt-in.

## The wedge
Mission control for parallel agents: detect which agent is blocked, surface the
actual question, and let you reply in one click — across every project.
